Mail.Ru Parking widget

Android widget to monitor Mail.Ru parking status.

Download .apk install file Download this project as a .zip file Download this project as a tar.gz file

What is it?

Recently @Mail.Ru Group has moved to a new office building with its own underground parking. Also the service to monitor free parking places was created as a web page. I decided to make an Android widget to monitor parking status on our mobile devices.

Features

  • Display free parking places with refresh date.
  • Process tap and double-tap on widget. By defaults to force update or to show details respectively.
  • Show last fetched data, timestamps of data update and refresh, fetched raw JSON.
  • Autoupdate data respecting internet availability. Battery is not draining while offline.

Also, there is useful feature called "Smart update". The update frequency varies depending on time of day:

  • On workdays widget updates once per hour.
  • On workdays within `hot hours' (from 9 till 12 am) widget updates each 15 minutes.
  • On holidays widget does not update at all.
